Regarding your request to turn the website design into a Figma file: as an AI, I cannot directly generate or export .fig files. However, there is a very easy way you can do this yourself!

You can use a free Figma plugin called html.to.design. Here is how you can do it:

  1. Open your Shared App URL in a new browser tab.

  2. Install the html.to.design browser extension (available for Chrome, Firefox, etc.).

  3. Click the extension while on your website to capture the page.

  4. Open Figma, run the html.to.design plugin, and import the file it generated.

This will instantly convert the live coded website into fully editable Figma layers!
